# ⚡ Circular Queue Visualizer

An interactive and visually appealing Circular Queue (Ring Buffer) visualizer built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🔁 Enqueue & Dequeue operations
- 👀 Peek (view front element)
- 🔄 Reset queue functionality
- 🎨 Neon-themed modern UI
- ✨ Smooth animations
- 📍 Highlights Front & Rear pointers
- ⚡ Real-time updates

---

## 🧠 Concepts Covered

- Circular Queue (Data Structure)
- Modulo Arithmetic
- Array Implementation
- DOM Manipulation
- Event Handling

---

## 💻 Tech Stack

- HTML
- CSS (Neon UI + Animations)
- JavaScript
